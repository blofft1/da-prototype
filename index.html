What changed vs. today's Browse view

1
Action menu, groupedBrowse / Search / Snapshots tabs are gone. Translation and MSM are now real screens (try clicking them). Inbox moved into the existing notification bell rather than adding a 5th button.
2
Inline AI omnibarCompact and contextual, not a separate destination. Also now the only way to create content — the old "+ Create" button is gone; that intent is AI-assisted.
3
Clickable breadcrumbMoved directly above the table. Clicking a segment re-populates the table in place (org's site list vs. this site's content) — no page navigation.
4
Name vs. TitleRestores the AEM concept of a URL-safe Name separate from a human Title.
5
Type badgeEvery row is labeled Page, Fragment, Config, Locale, or Asset instead of looking like an undifferentiated file list — and it's filterable via the chips above the table. Folder/file icons stay plain black and white; the Type column carries the signal.
6
StatusPublished / In review / Draft is visible inline — this is what "Snapshots" means today, surfaced without a separate app.
